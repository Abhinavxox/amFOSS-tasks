Review of AMFOSS tasks:

April 3, 2022
Till here the journey has been amazing. All 11 main tasks given by amfoss were really challenging. I'm already excited that if i get to be a member, I will learn so much from the club. The tasks were based on different languages like python, C, html, etc. From the date of getting these tasks till today, I've learned a lot and hope to learn more.

First task was to dual boot linux and windows. I was new to linux but the following task for getting used to terminal and its commands was amazing. I always wanted to learn terminal commands properly, but there is a thing like if you dont have to do something, you wont probably do it. After those tasks i learned a bit more about linux and why its different. But while using linux i dont just copy paste using the GUI but the terminal as it is more fun.

Then there was hackerrank problems. I had always heard of it but never tried. I didn't complete all but i tried to do as much as i could. But thanks to the tasks i got introduced to how everything works there. Then there was python based cli app. I come from java background. I was fascinated how with such short code we can do so much in python and the libraries are so useful. Then there was a task to correct the program written in C. I was a bit lost there so i straight jumped to CS50 and luckily they taught C in there. So I completed the CS50 tasks and C related task very easily. Then the flutter task, rich scrapper all were really fun.

I enjoyed all the tasks and followed every instructions. I didn't get completely into learning a different language but just the code and how things worked. Overall these tasks are great way to know test one's capability as well as intrest one in different fields included there.

Over and out...
